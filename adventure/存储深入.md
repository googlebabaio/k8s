# k8s的卷使用步骤

![](assets/markdown-img-paste-20200421120714336.png)
4个步骤：
provision 分配卷
attach 挂载卷
publish 创建文件系统
mount 将卷映射到容器的文件系统中

涉及到的核心组件2个：
- controller-manager
  - attach/detach controller 用于挂载卷和卸载卷
  - pv/pvc controller 用于创建和删除卷
- kubelet
  - Pod manager 他会去调用内置的volumn plugin组件
  - volumn plugin 实现了很多存储厂商实现的具体的存储的driver
